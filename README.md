## Используемые технологии

- React — компонентный подход  
- TypeScript — строгая типизация  
- Redux Toolkit — управление состоянием  
- React Router — маршрутизация  
- CSS Modules — изолированные стили  
- Storybook — разработка UI-компонентов

## Описание проекта

Веб-приложение для сборки бургера из ингредиентов с возможностью оформления заказа.  
Проект разработан в рамках обучения **Яндекс Практикум**.

Реализовано:
- Конструктор бургера (drag & drop);
- Авторизация и регистрация;
- Лента заказов;
- Модальные окна;
- Защищённые маршруты;
- Работа с API.

---

## Как запустить проект локально

1. Клонируйте репозиторий:
```bash
git clone https://github.com/maryamgab/frontend_yandex_stellar-burgers.git
cd frontend_yandex_stellar-burgers
```

2. Установка зависимостей:
```bash
npm install
```

3. Запуск проекта:
```bash
npm start
```
